# exercise-recovery

一个基于真实病例（术后恢复 + 体重管理）整理的**康复训练与生活方式计划**，目前以一个简单的静态页面形式呈现（`index.html`），方便在手机/电脑上随时查看“今日计划 / 整周计划”。

> 免责声明：本仓库内容仅用于健康管理与康复训练的整理与自我追踪参考，不替代医生诊断与治疗建议。如出现剧烈头痛、恶心呕吐、癫痫发作等症状，请立即就医。

## 目前内容

- `index.html`：康复减重计划看板
  - 自动显示当天（按系统日期）计划
  - 支持切换「查看整周 / 回到今日」
  - 包含：饮食建议、运动建议、久坐提醒、术后钻孔处瘙痒应对提示

## 快速开始

### 本地打开

直接用浏览器打开即可：

- 双击 `index.html`
- 或命令行：

```bash
# macOS
open index.html

# Linux
xdg-open index.html

# Windows (PowerShell)
start .\index.html
```

### 用本地 HTTP 方式打开（可选）

有些浏览器对本地文件的资源访问有限制，想更“像网页”可以起一个本地服务：

```bash
python3 -m http.server 5173
```

然后访问：<http://localhost:5173>
